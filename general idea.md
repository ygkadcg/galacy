玩家、敌机、子弹，都可以抽象成同一种对象，暂定类名为 `Obj`，包含如下属性与方法：

1. `int x, y;` 位置属性，单位为像素；
2. `enum type;` 类型属性，可以用枚举（或其他方法）标记当前对象的类型（玩家或敌机或子弹） 
3. `void move(const int dx, const int dy);` 方法，单位为像素，用以移动。

由于对象的长宽大小属性大多重复，故选择作为外部常量存储，再通过每个对象的具体类型（`type`）推导，而不是每个对象都存储长宽大小。

外部需要如下必要函数：

1. `bool collision(cosnt Obj *a, const Obj *b);` 判断两个对象是否相碰撞。

初始化如下：

1. 定义各类常量（如玩家、敌机、子弹的长宽，游戏刷新频率常量 `FPS` 等）；
2. 游戏界面初始化（如导入图片等）；
3. 创建玩家对象 `Obj player`；
4. 创建两个链表用以储存敌机、子弹。

游戏每 `FPS` 秒更新一次，每次更新如下：

1. 通过读取键盘输入获取玩家位置增量 `dx, dy`，以及是否发射新子弹；
2. 若需要，生成新敌机、新子弹（链表尾部增加）；
3. 对所有对象（玩家、敌机、子弹）调用 `move()` 方法进行移动；
4. 遍历链表中所有敌机与子弹，用 `collision()` 函数判断碰撞，若需要，删除敌机（链表删除）或结束游戏；
5. 绘制图形化界面（这部分估计得大量CV）。
